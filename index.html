Built with Blender.

All of the models used in this assignments have been created using blender. They have been created from scratch and are original models and have not been take from online.

Building with blender

Building in Blender has been easily due to the similarity is has to Cinema4D a software which I have used before for different projects for the past year.

To create both the bottle and the can, I started with a refrence image which would used to create the basic shape. The wireframe helped to use the whole cylindrical object to make sure that the overall shape of the model is consistent.

In order the make the shape more realistic, using multiple small wire cuts to make minor changes to the shape. This means when the object is given subdivision surface it will appear smoother in comparison to an object with siminal loop cuts.
